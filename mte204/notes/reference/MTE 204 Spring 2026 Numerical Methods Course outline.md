Course Description
Calendar Description for MTE 204
Number systems and machine errors; roots of non-linear equations; matrix calculations; Eigenvalue and Eigenvector
calculations; interpolation and approximation; numerical integration and solution of ODE's (linear and non-linear) and
systems of ODEs; calculation of series; solution methods for PDE's; use of numeric and symbolic computing tools.

Learning Outcomes
By the end of this course students should be able to...
1. Develop and implement stable and accurate numerical methods to solve linear
systems of equations, find root of equations and perform curve fitting, numerical
integration and differentiation [GA: KB (I)]
2. Understand the basic concepts of partial differential equations, including initial and
boundary conditions and analytical solution techniques [GA: KB (I)]
3. Use numerical methods, including finite difference and finite element analysis, to
solve ordinary differential equations and partial differential equations for
engineering problems [GA: KB (I)]
(GA: Graduate Attributes, KB: Knowledge Base)
